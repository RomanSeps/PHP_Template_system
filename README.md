# PHP_Template_system 💻
Hello! 👋 Welcome to the template system! <br>

## Description 📄
The purpose of this repository is to serve as an example of the basic use of templates in PHP.
Loading data into a file dynamically, not having to rely on tens or even hundreds of files for a website.
Everything is connected, there's no need for multiple css / js files... <br>
<br>
Want to change a style of something? Just change up style.css, and the style changes across ALL your files.

### P.S. ✉️
This is meant for my own personal use, but feel free to use it too, in case you find it valuable (probably not the most bug-friendly / effective template system out there :D ). It's also meant for me to practice basic file structure ;)
